쉴물가 보고서 대시보드 사용 가이드

목사님 사역을 돕는 스마트 모임 현황 & 심방 관리 대시보드 안내서

✨ 대시보드 4대 핵심 기능

🌿

모임 실시 현황 자동 집계

선택한 초장 및 조회 기간 내에 실제 모임을 실시한 쉴물가(목자) 수와 전체 보고서 제출 건수를 자동으로 보여줍니다.

🚨

AI 스마트 심방필요 모아보기

입원, 수술, 장기결석, 우울, 위기, 부고 등 돌봄이 필요한 소식을 지능적으로 감지하여 버튼 하나로 모아서 확인합니다.

📌

담당 초장 고정 (핀 기능)

목사님이 담당하시는 초장(예: 1초장)을 핀 버튼으로 저장하면 접속 시 자동으로 해당 초장 보고서부터 펼쳐집니다.

📱

스마트폰 앱(PWA) 추가 지원

별도 앱설치 없이 아이폰/안드로이드 홈화면에 '목자 대시보드' 앱으로 등록하여 언제 어디서나 즉시 조회할 수 있습니다.

📌 2. 내 담당 초장 선택 및 📌 핀 고정

1

담당 교구(초장)를 선택 후 📌 버튼을 누르세요

초장 선택창에서 본인의 담당 초장(예: 1초장, 2/3초장 등)을 선택한 뒤 바로 옆의 📌 핀 버튼을 누르면 첫 화면 설정이 완료됩니다.

2

로그인 시 자동으로 내 초장이 조회됩니다

이후에는 대시보드에 접속하실 때마다 번거롭게 필터를 바꾸실 필요 없이, 목사님의 담당 초장 보고서가 자동으로 가장 먼저 나타납니다.

🔍 3. 보고서 카드 상세 보기 및 주요 모임 내용

1

보고서 카드를 터치/클릭하면 내용이 펼쳐집니다

목자 이름, 모임 날짜, 장소(예: 1-7 교실)가 요약 표시되어 있으며, 클릭하면 참석자/불참 사유/소감/기도제목이 확장 표시됩니다.

2

노란색 하이라이트로 긴급 키워드가 강조됩니다

기도제목이나 불참 사유에 입원, 수술, 장기결석, 우울 등의 중요한 단어가 포함된 경우 시스템이 시각적으로 즉시 알아볼 수 있게 강조합니다.

📲 4. 스마트폰 홈화면에 앱(PWA) 추가하는 방법

스마트폰 웹 브라우저로 대시보드 주소에 접속하신 후, 아래 절차대로 홈화면에 추가하시면 어플리케이션처럼 편리하게 바로 실행할 수 있습니다.

아이폰 (Safari 브라우저)

  1. Safari로 대시보드 페이지 접속
  2. 하단 중앙의 공유 버튼 (Square with Arrow) 터치
  3. 메뉴를 아래로 내려 [홈 화면에 추가] 선택
  4. 우측 상단 [추가] 터치 완료

안드로이드 (Chrome 브라우저)

  1. Chrome으로 대시보드 페이지 접속
  2. 우측 상단 점 3개 메뉴 (⋮) 터치
  3. [앱 설치] 또는 [홈 화면에 추가] 선택
  4. 팝업 창에서 [추가] 터치 완료

📝 참고: 쉴물가 목자용 보고서 작성 시스템 (`index.html`)

목자님들께 공유되는 제출 링크는 스마트폰 최적화 폼으로 구성되어 있습니다.
- **쉴물가 자동완성**: 초장을 선택하고 이름을 입력하면 등록된 쉴물가명이 자동 추천됩니다.
- **참석자 자동 계산**: 성함을 쉼표(,)로 작성하면 인원수가 자동 계산됩니다.
- **모임 내용 칩 버튼**: 말씀나눔, 기도나눔, 교제, 식사 등을 손가락으로 가볍게 터치하여 제출합니다.
